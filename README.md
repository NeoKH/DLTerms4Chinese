# DLTerms4Chinese

很多翻译工具逐渐支持添加术语库，比如DeepL、有道词典9、百度翻译开放平台等。
创建本仓库的目的是收集深度学习术语的恰当的中文翻译，以改善翻译软件在深度学习方面的准确率。
